### 회원

- [x] 회원가입 할 수 있다.
  - [x] 회원 ID는 4자 이상 12자 이하여야한다.
  - [x] 회원 PASSWORD는 4자 이상이어야 한다.
  - [x] 회원 PASSWORD는 숫자, 영어, 특수문자 조합이어야한다.
  - [x] 회원 NICKNAME은 2자 이상 10자이하여야 한다.
- [x] 로그인할 수 있다.
